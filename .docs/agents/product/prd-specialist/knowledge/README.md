# prd-specialist Knowledge Base

This directory stores accumulated knowledge and learnings for the prd-specialist agent.

## Purpose

Knowledge files capture patterns, insights, and best practices learned from PRD creation sessions:
- Common product patterns by domain
- Effective persona structures
- Success metric patterns
- Constitutional customization patterns
- User research insights

## Files

Knowledge accumulates over time and may include:
- Product domain patterns (e.g., e-commerce, SaaS, marketplace)
- Persona templates that work well
- Success metric formulas by industry
- Common pitfalls and how to avoid them
- Effective questioning techniques

## Usage

The prd-specialist agent uses this knowledge to:
- Provide domain-specific guidance
- Suggest relevant patterns and examples
- Improve questioning strategies
- Share proven approaches
- Avoid previously identified anti-patterns
