# prd-specialist Performance Metrics

This directory stores performance tracking and optimization data for the prd-specialist agent.

## Purpose

Performance files track the effectiveness of PRD creation:
- Time to complete PRD sections
- Number of clarification rounds needed
- PRD approval rates
- Downstream impact (feature velocity, spec clarity)
- User satisfaction metrics

## Files

Performance data may include:
- Session duration statistics
- Iteration counts per section
- Approval/rejection rates
- Quality metrics (completeness, clarity, actionability)
- User feedback and ratings

## Usage

The prd-specialist agent uses performance data to:
- Identify improvement opportunities
- Optimize questioning approaches
- Benchmark PRD quality
- Track effectiveness over time
- Validate changes to methodology
