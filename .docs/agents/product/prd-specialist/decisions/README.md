# prd-specialist Decision Log

This directory stores historical decisions and rationales for the prd-specialist agent.

## Purpose

Decision files document key choices made during PRD creation:
- Constitutional customization decisions
- Scope prioritization rationales
- Feature-to-MVP assignments
- Risk assessment decisions
- Success metric selections

## Files

Decision logs capture:
- What decision was made
- Why it was made (rationale)
- What alternatives were considered
- Context and constraints
- Outcome (if known)

## Usage

The prd-specialist agent uses decision logs to:
- Maintain consistency across related PRDs
- Explain past choices to stakeholders
- Learn from previous decision outcomes
- Avoid repeating mistakes
- Document tradeoffs for future reference
